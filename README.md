 The objective of this project is to create a simple shell.

![Untitled](image/Untitled.png)

## Compile & Run

```bash
#1) compile Lee-shell
git clone https://github.com/kimminkyeu/Lee-shell && cd ./Lee-shell && make

#2) run shell program
./lesh   
```
